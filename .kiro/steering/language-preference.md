# 語言偏好設定

請在所有回應中使用**繁體中文**，除非：
- 程式碼註解和變數命名（使用英文）
- 用戶明確要求使用其他語言
- 技術術語保持英文（如 JWT、API、ORM 等）

## 回應風格

- 使用台灣常用的技術用語
- 保持專業但友善的語氣
- 程式碼範例使用英文註解
- 文件說明使用中文

## 範例

✅ 正確：
```typescript
// Fetch user profile
const user = await db.select().from(users)
```
這段程式碼會從資料庫取得使用者資料。

❌ 避免：
All responses in English unless requested.
